Desktop Environment Selection

Press @@@fkey_de@@@ to get the list of desktop environment included by MagOS-Linux. There are three DE:

1. KDE is the most user freindly DE. KDE system requirements are: Processor 1GHz (x86), Memory 756 MB This is default DE.

2. LXDE is lightweight and fast DE. System requirements are: Processor 600MHz (x86), Memory 256 MB

3. You can launch Steam client in big picture mode as a DE. This allows you to use your hardware as a gaming console.

Also you may boot system without any DE in textmode console.
