Other parameters

There are two types of boot options available. First - options that affect the system. Second - kernel options. You can use both types in bootloader command line. There are some samples, that may be usefull for you.

Copy to RAM (copy2ram uird.copy2ram freemedia uird.freemedia). System will be copied to RAM and will be started from memory. This requires more than 2GB RAM. It is possible to detach media when system will be loaded into RAM and started.

Video safe setting (no3ddesktop xdriver=fbdev). Video failsafe parameters . Select this if you encounter problems while starting graphical desktop. This option forces to use very similar video driver fbdev and switches off compiz-fusion effects.

No modules (noload=/modules/ uird.noload=/modules/). Do not load external modules from /modules/ folders. If you installed external modules and it made you system unfunctional you can disable ones there.

Flight mode (rfkill.default_state=0 flightmode). Turn off wireless devices. You can disable wifi, bluetooth, etc devices there.
