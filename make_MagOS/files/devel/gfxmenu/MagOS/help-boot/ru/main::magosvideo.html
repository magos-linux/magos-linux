Выбор разрешения

Нажмите @@@fkey_videom@@@, чтобы получить список режимов видеокарты. Выбранный режим будет использован при загрузке системы. Обратите внимание, что данное разрешение экрана влияет только на режим загрузки системы. Разрешение графического сервера xorg не зависит от данного параметра

По умолчанию установлено разрешение 1024х768. Если ваш монитор или видеокарта не поддерживает данный режим, выберите желаемое разрешение вручную.

Если ваша система имеет проблемы с видеокартой во время загрузки, может пригодиться текстовый режим.
